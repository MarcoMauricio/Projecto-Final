
To run the project:

	1. Configure the settings.config file on both projects in the "Presentation" solution folder.
	   Specially the urls if you change the project build configuration.

	2. Configure the connections.config file in the EggOn.Web.Service project with your SQL Server
	   (Normal/Express or Localdb) connection string.
